Descrição Detalhada

View objects usados para a passagem de dados entre a camada de visualização e a de negócios.

Por padrão, todos os VOs implementam java.io.Serializable, para que possam trafegar pela rede. A camada de negócios preenche os VOs com dados para exibição nas páginas web, enquanto a camada de visualização envia dados inseriods/alterados pelo usuário em VOs para serem processados pela camada de negócios.
